# C++17 STD

> Test build example using CMake

## Build Makefile

```sh
$ mkdir build && cd build && cmake ..
```

## Make Doxygen documentation

```sh
$ make doc
```

## Make & run application

```sh
$ make cpp17std && ./cpp17std
```

## Make & run googletests

```sh
$ make cpp17std-test && ./cpp17std-test
```
